# 羽毛球爱好者网站

一个专业的羽毛球知识分享网站，包含基础知识、技巧教学和装备推荐。

## 功能特点

- 🏸 **基础知识**：详细的比赛规则、计分方法和发球规则
- 🎓 **技巧教学**：正手击球、反手击球、基本步法等专业技巧
- 🛍️ **装备推荐**：球拍、羽毛球、运动鞋等装备的选择指南
- 📱 **响应式设计**：适配手机、平板和桌面设备
- 🎨 **现代化UI**：简洁美观的界面设计

## 文件结构

```
├── index.html      - 主页面文件
├── styles.css      - 样式文件
├── script.js       - JavaScript交互逻辑
└── README.md       - 项目说明文档
```

## 使用方法

1. 直接双击 `index.html` 文件在浏览器中打开
2. 或者使用本地服务器：
   ```bash
   python -m http.server 8000
   ```
   然后访问 `http://localhost:8000`

## 自定义修改

您可以根据需要修改以下内容：

1. **颜色主题**：编辑 `styles.css` 中的 CSS 变量
2. **内容**：编辑 `index.html` 中的文字和链接
3. **样式**：修改 `styles.css` 中的样式规则
4. **交互**：修改 `script.js` 中的 JavaScript 代码
